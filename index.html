⚫⚪ Weiqi Assets

围棋静态资源库 - 二维码分享、棋谱编解码

📦 资源列表

  • share.js - 核心分享库
    https://weiqi-dev.github.io/weiqi-assets/js/share.js
    提供二维码生成、棋谱编码解码功能
  • share/ - 扫码下载页
    https://weiqi-dev.github.io/weiqi-assets/share/
    扫码后自动下载 SGF 棋谱文件

🚀 快速开始

在网页中引入分享库：

<script src="https://weiqi-dev.github.io/weiqi-assets/js/share.js"></script>

生成二维码：

WeiqiShare.generateQR(container, gameState) .then(() => console.log('成功')) .catch(err => console.error(err));

📖 API 文档

方法 说明
isWechat() 检测微信浏览器
encode(gameState) 编码棋谱为紧凑格式
decode(base64) 解码 base64 为棋谱数据
toSGF(data) 棋谱数据转 SGF 格式
generateQR(container, gameState) 生成二维码图片
